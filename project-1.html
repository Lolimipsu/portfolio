Project Overview

In this project, I led the development of an employee retention prediction model utilizing the Random Forest Classifier, this research was conducted using the IBM HR dataset sourced from Kaggle and involved collaboration with an HR specialist to ensure practical relevance.

This research project aimed to address the critical issue of employee retention by leveraging machine learning techniques. The project commenced with sourcing data from the IBM HR dataset on Kaggle, which offered comprehensive insights into employee demographics, performance, and turnover.

To enhance the usability and accessibility of our system, we developed an interactive web application. This webapp provides users with an intuitive interface to interact with the model, explore various metrics, and gain insights into the factors driving employee retention. Key features of the webapp include visual representations of the model's predictions, such as bar graphs, box plots, univariate analysis (KDE plot, violin plot), confusion matrix, correlation heatmap, and variable importance plots.

By integrating technical rigor with practical insights from an HR specialist, this project not only demonstrates the potential of machine learning in tackling real-world business challenges but also provides a valuable tool for organizations to better understand and address employee retention.
